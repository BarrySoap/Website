Previous Work

During my time at University, I've tried my best to make work which I could show to others - whether this be a game, graphical project or general piece of software. Games which I've created in my spare time can be found on my Global Game Jam page. Miscellaneous software projects and/or courseworks can be found on my GitHub page.

The global game jams are annual events which brings game developers together to create games in small groups within 48 hours. I've took part in 4 of those events, and I feel as though my quality of content is good. For the first year I helped create a dungeon crawler game in which the protagonist is trying to escape the multiple layers of hell. The game had different levels, different enemies, abilities and a boss. For the subsequent years I created a mining game myself, a small-scale JRPG and a thievery game.
