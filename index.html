Previous Work

During my time at University, I've tried my best to make work which I could show to others - whether this be a game, graphical project or general piece of software. Games which I've created in my spare time can be found on my Global Game Jam page. Miscellaneous software projects and/or courseworks can be found on my GitHub page.

The global game jams are annual events which brings game developers together to create games in small groups within 48 hours. I've took part in 3 of those events, and I feel as though my quality of content is somewhat decent. For the first year I helped create a dungeon crawler game in which the protagonist is trying to escape the multiple layers of hell. The game had different levels, different enemies, abilities and a boss. For the subsequent years I created a mining game myself and then co-created a JRPG-style game.

For the courseworks I've created during my time at university, they range from:

  • Graphics Programming using OpenGL and GLEW.
  • Bash Shell Scripting.
  • Video game engine creation using C++.
  • Parallel processing using Groovy.
  • A functional game of checkers using C# WPF documents.
  • A photorealistic mobile skiing game for Disney.
